# Dopax Logo

Primary files:

- `dopax-primary.svg`: horizontal logo with icon, wordmark, and descriptor
- `dopax-mark.svg`: icon-only mark for app icon, favicon, and avatar use

Design direction:

- The outer `D` frame gives the brand a stable, product-led silhouette.
- The crossed inner strokes hint at the `x` in `dopax` and suggest routing, synthesis, and decision-making.
- The warm orange-to-amber gradient keeps it human and energetic instead of generic cold-tech branding.
- The deep ink accent adds trust and contrast for UI use.

Core colors:

- `#F36A42`
- `#FFB13C`
- `#122033`
- `#586676`
